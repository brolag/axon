# Hero asset

Drop the hero image here as `hero.png`.

The landing hero reserves a band for it (`object-fit: cover`, masked into the
cream background at low opacity so the text stays legible). If `hero.png` is
absent, the page falls back to a CSS watercolor gradient with the same masking,
so the build is valid with or without the file.

Suggested: a wide, soft watercolor in the Indie Mind palette (teal / soft blue /
soft green) with the neural-spiral motif. Recommended aspect roughly 3:1,
1600px wide or more.
